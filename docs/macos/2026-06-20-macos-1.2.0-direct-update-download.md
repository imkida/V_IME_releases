# VIME macOS 1.2.0 beta

这是 VIME macOS 1.2.0 beta 的维护包，建议正在使用 1.2.0 测试版的用户更新。

- 设置页「关于」里的「版本更新」在发现新版本时会显示更明确的「下载 DMG」入口。
- 同一状态行保留「查看更新说明」，用户可以先看更新内容，也可以直接下载更新包。
- 如果更新清单临时缺少安装包地址，会降级为「打开下载页」，避免用户没有下一步。
- 继续保留登录时自动启动、Dock 图标设置、备份恢复、权限排障和默认云端服务。
- 默认云端服务仍通过 VIME 代理提供，包内不包含真实 provider key。

下载 DMG 后打开，把 VIME 拖到 Applications；如提示已存在，请选择「替换」。安装完成后，从 Applications 打开 VIME。

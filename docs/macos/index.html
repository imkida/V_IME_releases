04

常见问题

遇到写入失败、权限未生效或网络不可用时，可以先按下面的方式排查。

提示辅助功能未授权
在系统设置中允许 VIME 使用辅助功能。授权后通常自动生效；如果长时间仍不可用，再完全退出并重新打开 VIME。
没有写入目标输入框
先确认光标在目标输入框内。若 VIME 无法安全写回，会保留结果并提示复制，避免说过的内容丢失。
默认服务不可用
请先检查网络。也可以在「服务配置」中保存自己的服务密钥，个人服务会优先使用。
右 Command 和外接键盘冲突
部分外接键盘、远程桌面或改键工具可能拦截右 Command。可以在设置中配置备用快捷键。

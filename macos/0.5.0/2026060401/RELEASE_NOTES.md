# V_IME macOS 0.5.0 beta (2026060401)

这是面向小范围内测用户的正式签名修正版。相比上一版，这次安装包已使用 Developer ID 签名，并完成 Apple notarization 与 DMG ticket stapling，推荐测试用户优先安装 DMG。

## 本次更新

- 安装包已完成 Apple notarization，降低首次打开时被 macOS 拦截的概率。
- 已启用 V_IME 默认云端服务：未配置个人 API Key 时，也可以使用语音识别和文本整理。
- 如果你保存了个人 API Key，V_IME 仍会优先使用你的个人服务配置。
- 保留上一版 HUD 提示体验优化：编辑模式未选中文本时显示轻量提示，错误提示更柔和，语音波形颜色更清晰。

## 使用说明

- 推荐下载并安装 DMG：`V_IME-0.5.0-2026060401-macos.dmg`。
- 如果你已安装旧版本，可以直接用新版覆盖安装。
- V_IME 是菜单栏应用。启动后如果没有出现普通窗口，请查看屏幕右上角菜单栏里的 V_IME 图标；如果开启了 Dock 图标，也可以从 Dock 进入设置。
- 如果系统提示需要授权，请在 macOS 系统设置中确认 V_IME 的麦克风、辅助功能和输入监控权限。
- 如果遇到无法写入目标输入框的情况，可以先使用 HUD 里的复制结果作为兜底，并反馈具体目标 App。

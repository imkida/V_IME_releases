# V_IME macOS 0.5.0 beta (2026060305)

这是一个面向小范围内测用户的修正版更新包，重点补回默认云端服务，并保留上一版 HUD 提示体验优化。

## 本次更新

- 已启用 V_IME 默认云端服务：未配置个人 API Key 时，也可以使用语音识别和文本整理。
- 如果你保存了个人 API Key，V_IME 仍会优先使用你的个人服务配置。
- 编辑模式未选中文本时，现在会显示轻量提示，不再弹出强错误提醒。
- HUD 错误提示改为更柔和的提示样式，减少红色警示感，并保留清晰的关闭或操作按钮。
- 语音输入时的声波颜色更明亮，录音状态更容易看清。

## 使用说明

- 这是测试版安装包，主要用于早期共测和问题收集。
- 如果你已安装旧版本，可以直接安装新版覆盖。
- 该包已使用 Developer ID 签名，但本次为 fast 包，未走 Apple notarization；如果 macOS 提示无法验证，请在"隐私与安全性"中允许打开，或右键选择打开。
- 如果系统提示需要授权，请在 macOS 系统设置中确认 V_IME 的麦克风、辅助功能和输入监控权限。
- 如果遇到无法写入目标输入框的情况，可以先使用 HUD 里的复制结果作为兜底，并反馈具体目标 App。
